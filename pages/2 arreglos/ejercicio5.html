Números pares de un arreglo recorridos desde el último al primero

codigo : javascript
//1. Declarar un arreglo de números ingresado por el usuario
let numeros = [10, 15, 23, 30, 45, 50];
//2. Recorre el arreglo desde el último elemento al primero
for(let i = numeros.length - 1; i >= 0; i--) {
//3. verifica Si el número es par.
if(numeros[i] % 2 === 0) {
console.log(numeros[i] +"es par");
}
}
