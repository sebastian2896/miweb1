Mayor de los elementos de un arreglo

codigo : javascript
//1. Declarar un arreglo de números ingresado por el usuario
let numeros = [10, 15, 23, 30, 45, 50];
//2.supone que el primer elemento es el mayor.
let mayor = numeros[0];
//3.Usar un bucle for para comparar cada elemento con el supuesto mayor e itera desde el segundo.
for (let i = 1; i < numeros.length; i++) {
//4.actualiza mayor si se encuentra un número más grande
if (numeros[i] > mayor) {
mayor = numeros[i];
}
}
console.log("El mayor número es " + mayor);
