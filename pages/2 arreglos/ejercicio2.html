Promedio de un arreglo

codigo : javascript

let numeros = [10, 15, 23, 30, 45, 50];
let suma = 0;
for (let i = 0; i < numeros.length; i++) { //1 itera a través del arreglo
suma += numeros[i]; //2 suma cada número al total.
}
let promedio = suma / numeros.length; //3 calcula el promedio.
console.log("El promedio es " + promedio);
}


Aquí se mostrarán el promedio
