Verificar si un número es primo

Un número es primo si solo tiene dos divisores: 1 y el mismo número.
//1. Declarar una variable para el número
let numero = 7;
let esPrimo = true;
//2. Usar un bucle for para encontrar los divisores
for(let i = 2; i < numero; i++) {
//3. Usar una condición if para verificar si tiene más de dos divisores if(numero % i === 0) {
esPrimo = false;
break;
}
}
if(esPrimo) {
console.log(numero + " es primo");
} else {
console.log(numero + " no es primo");
}
