Invertir una cadena

codigo : javascript
//1.Declarar las variables cadena y cadenaInvertida.
let cadena = "Hola Mundo";
let cadenainvertida="" //2.Usar un bucle for que empieza desde el último carácter (cadena.length - 1) y decrementa i hasta 0.
for (let i = cadena.length - 1; i >= 0; i--) {
//3.En cada iteración, añadir el carácter actual a cadenaInvertida.
cadenaInvertida += cadena[i];
}
console.log(cadenaInvertida);
